Dynamic API Endpoints

GET /api/tables List all discovered tables
GET /api/tables/{table}/schema Get table schema and columns
GET /api/tables/{table}/records List records with pagination
GET /api/tables/{table}/records/{pk} Get single record by primary key
GET /api/tables/{table}/search?q=term&columns=col1,col2 Multi-column search (ClickHouse accelerated)
GET /api/tables/{table}/stats Get table statistics (estimated count)

ClickHouse Setup (for 2TB+ databases)

To enable ClickHouse-accelerated search, set these environment variables:

CLICKHOUSE_ADDR=localhost:9000
CLICKHOUSE_DATABASE=default
CLICKHOUSE_USERNAME=default
CLICKHOUSE_PASSWORD=your_password
CLICKHOUSE_SYNC_TABLES=table1,table2,table3
CDC_SYNC_INTERVAL_SECONDS=30
CDC_BATCH_SIZE=1000

The API will automatically sync data from PostgreSQL to ClickHouse and use it for fast multi-column text searches.
